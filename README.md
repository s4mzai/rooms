# rooms

A private, self-destructing chat room application built with Next.js, Upstash Realtime & Redis, and Elysia (server API).

## Features

- Create ephemeral chat rooms that auto-expire.
- Real-time messaging powered by Upstash Realtime.
- Message history and room metadata stored in Upstash Redis.
- Self-destruct (destroy) rooms and messages programmatically.
- Minimal, client-first UI built with React/Next.js and Tailwind CSS.

## Stack

- Next.js 16 (React 19)
- TypeScript
- Elysia + @elysiajs/eden (internal API treaty client)
- Upstash Realtime + Upstash Redis
- React Query (@tanstack/react-query)
- Tailwind CSS
